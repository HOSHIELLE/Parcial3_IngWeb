# Formulario de Inscripción – Parcial 3  
Sistema web desarrollado como parte de un proyecto académico, enfocado en la recolección y gestión de inscripciones utilizando **PHP**, **MySQL** y una interfaz moderna personalizada en tonos morados.

## Descripción  
Este proyecto consiste en un formulario web donde los usuarios pueden registrarse ingresando sus datos personales y seleccionando los temas tecnológicos de su interés.  
El sistema está construido con una arquitectura sencilla y clara:
- Frontend con HTML5 y CSS3 personalizado  
- Backend en PHP utilizando PDO  
- Base de datos MySQL estructurada con relaciones muchos-a-muchos  
El objetivo principal es practicar validaciones en PHP, manejo de formularios, transacciones seguras y generación de reportes dinámicos.

## Características  
- **Formulario completo:** nombre, apellido, edad, sexo, país, nacionalidad, correo, celular y observaciones  
- **Temas tecnológicos con checkbox:** selección múltiple  
- **Validaciones en PHP:** control de datos y mensajes de error  
- **Inserción segura en MySQL:** uso de transacciones y sentencias preparadas  
- **Reporte dinámico:** listado completo con unión de tablas y `GROUP_CONCAT`  
- **Diseño personalizado:** estilo morado con interfaz estética y clara  
- **Código completamente comentado:** ideal para fines educativos

## Tecnologías Utilizadas  
### Frontend  
- HTML5

### Backend
